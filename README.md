# IAcriolla 🧉💻

**A "cyber-criollo" laboratory for Local AI, Multi-Agent systems, and Cognitive Psychology experiments.**

This is where I build, break, and optimize tools at the intersection of human cognition and local-first artificial intelligence. 


## 🌪️ Current Experiment: Typhon Stress Test

**Typhon** is an Open Source testing framework designed to evaluate the resilience and performance of **Local AI** environments.

* **Focus**: Stress-testing local LLMs (Hermes, Llama.cpp)
* **Goal**: Ensuring stability and privacy in high-demand scenarios.

[Explore Typhon Stress Test →](https://github.com/IAcriolla/typhon-stress-test)


## 🔬 Research Areas

* **Local-First AI**: Experimenting whatever runs on a 3090 on private infrastructure.
* **Multi-Agent Optimization**: Identifying bottlenecks in agent-to-agent communication.
* **Backpropagating Ψ into AI**: Integrating psychological principles into IA and agent design.
